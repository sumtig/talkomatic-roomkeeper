# talkomatic-roomkeeper
this talkomatic bot keeps any room joined alive until its exited. cant get kicked
# how to use
just run the file, please make sure to get a different token from https://github.com/Talkomatic-CLI-Edition.
1. install the python file
2. run it
3. get the token from the generated file (tk_a bunch of numbers and letters, youll get it)
4. paste it into the token field.
5. done, run it and enter a room code. ittl join
# prerequisites
you need node.js for it to work. do `npm install` in the folder.
